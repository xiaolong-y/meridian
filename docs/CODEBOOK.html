Notes

  1. FRED is the primary source for US economic data, providing actual reported values with monthly/quarterly frequency. Requires a free API key.
  2. IMF DataMapper provides forecasts through 2030. The "US Outlook" section shows these projections for comparison with current FRED data.
  3. ECB Policy Rate shows the Deposit Facility Rate, which is the rate banks receive for overnight deposits with the ECB. This is the key rate used to implement monetary policy since 2022.
  4. World Bank Real Interest Rate is calculated as: lending rate minus inflation (GDP deflator). Data typically lags 1-3 years.
  5. Japan e-Stat data is sourced from official Japanese government statistics (Statistics Bureau, Cabinet Office, METI).
  6. Sparklines display the last 16 data points using Unicode braille patterns for 2x resolution.
  7. Change values show period-over-period change where available (e.g., QoQ for quarterly data, MoM for monthly).

API Endpoints

FRED:       https://api.stlouisfed.org/fred/series/observations?series_id={id}&api_key={key}
IMF:        https://www.imf.org/external/datamapper/api/v1/{indicator}/{country}
ECB:        https://data-api.ecb.europa.eu/service/data/{dataflow}/{series_key}
World Bank: https://api.worldbank.org/v2/country/{country}/indicator/{indicator}
e-Stat:     https://dashboard.e-stat.go.jp/api/1.0/Json/getData?IndicatorCode={code}
Yahoo:      https://query1.finance.yahoo.com/v8/finance/chart/{ticker}
CoinGecko:  https://api.coingecko.com/api/v3/coins/{id}/market_chart
HN:         https://hacker-news.firebaseio.com/v0/topstories.json
Algolia:    https://hn.algolia.com/api/v1/search?query={query}

FRED API Setup

To enable FRED data fetching:

  1. Get a free API key from fred.stlouisfed.org
  2. Set the environment variable: export FRED_API_KEY=your_key_here
  3. Run the dashboard updater: python -m src.main
